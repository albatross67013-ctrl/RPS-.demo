# Rock paper scissor

A simple command line rock , paper, scissor game built with python challenge the computer which make random choice and see if you can win

FEATURES 

* play agianst the computer
* Rondom computer moves
* Easy result ( win ,lose, and draw)
* Easy python code
* Biggeners friendly project

TECHNOLOGY USED

* PYTHON

PROJECT STRUCTURE

* Rocks - paper - scisssors

# HOW TO PLAY

* COMPUTER CHOICE IS RANDOM
* YOU JUST NEED TO SELECT YOUR CHOICE
* THE WINNER IS DECIDED BASED ON THE CLASSIC RULES:
  * ROCKS BEAT SCISSORS
  * SCISSORS BEAT PAPER
  * PAPER BEAT ROCK
